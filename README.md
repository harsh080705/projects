# 💱 Currency Converter Web App
A sleek, responsive web application that provides instant currency conversion using real-time market data.

## 🚀 Features
*   **Real-time Rates:** Fetches live data from [Currency Converter, e.g., ExchangeRate-API].
*   **Country Flags:** Automatically updates flags based on the selected currency.
*   **Reverse Conversion:** Easily swap 'From' and 'To' currencies with one click.
*   **Mobile First:** Fully responsive design for seamless use on any device.

## 🛠️ Tech Stack
*   **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
*   **API:** [ExchangeRate-API] for fetching latest exchange rates.
